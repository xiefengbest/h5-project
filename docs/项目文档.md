# AnyWP H5 项目文档

## 项目概述

AnyWP 是 **产品经理 / 设计用高保真原型**，演示 TCL 锁屏杂志相关界面与交互。纯静态 HTML，本地打开 `pages/` 即可预览，**不面向线上部署**。

**源码目录** `pages/`：

| 目录 | 说明 |
|------|------|
| `wallpaper/` | 壁纸 App + 详情页短剧入口 Demo【手机】 |
| `minus-one/` | 负一屏 v1（news）/ v2（discover）【手机】 |
| `web-anyselected/` | 资讯网站完整 Demo / 新闻详情【手机】 |
| `pages-pad/` | **Pad App 独立原型目录**（1280px，与 `pages/` 平级），内含 wallpaper / minus-one / web-anyselected |

Pad 版与手机版功能一致，布局按 Android 平板规范适配，对应研发计划中的独立 Pad App。Pad 内页面跳转仅在 `pages-pad/` 目录内闭环。**Pad 版暂不包含模拟广告位**（待 Pad 广告策略确定后再设计）。

Pad 壁纸横竖屏内容策略见 [pad-壁纸横竖屏内容策略.md](./pad-壁纸横竖屏内容策略.md)。

**页面跳转**：见 [页面跳转关系图.md](./页面跳转关系图.md)。

## 页面结构

### 1. 壁纸 App (`pages/wallpaper/index.html`)

| 界面 | 触发 |
|------|------|
| 首页瀑布流 | App 主动打开 |
| 详情页 `#detail-page` | 锁屏左滑 / 首页点壁纸 |
| 全屏预览 `#preview-page` | 详情页点主图 |

详情页资讯入口 → `../web-anyselected/index.html?id=N`

### 2. 详情页短剧入口 (`pages/wallpaper/short-dramas.html`)

模拟壁纸详情页中增加的短剧模块，含列表与全屏播放器。

### 3. 负一屏 v1 (`pages/minus-one/news.html`)

News 列表 + 游戏横滑 + 广告。点击新闻 → `../web-anyselected/index.html?id=N`

Pad 版：`pages-pad/minus-one/news.html` → `../web-anyselected/index.html?id=N`

### 4. 负一屏 v2 (`pages/minus-one/discover.html`)

Discover Widget 聚合。骨架屏：`discover-skeleton.html`

Pad 版：`pages-pad/minus-one/discover.html`

### 5. 资讯网站 (`pages/web-anyselected/index.html`)

网页端完整 Demo。单页多态：列表 / 详情 / 说明页 / 多语言。导读：[h5-README.md](./h5-README.md)

Pad 版：`pages-pad/web-anyselected/index.html`（列表双列、详情主从双栏）

## 日期格式化规则

### 新闻日期展示规则（国际通用）

```javascript
function formatDate(date, lang = 'en') {
    const now = new Date();
    const target = new Date(date);
    const diffMs = now - target;
    const diffSec = Math.floor(diffMs / 1000);
    const diffMin = Math.floor(diffSec / 60);
    const diffHour = Math.floor(diffMin / 60);
    const diffDay = Math.floor(diffHour / 24);

    // 1 分钟内：Just now
    if (diffSec < 60) {
        return lang === 'zh' ? '刚刚' : 'Just now';
    }

    // 1 小时内：Xm ago
    if (diffHour < 1) {
        return `${diffMin}m ago`;
    }

    // 24 小时内：Xh ago
    if (diffDay < 1) {
        return `${diffHour}h ago`;
    }

    // 7 天内：Xd ago
    if (diffDay < 7) {
        return `${diffDay}d ago`;
    }

    // 同年内：月 日（如 Jan 15）
    if (target.getFullYear() === now.getFullYear()) {
        return target.toLocaleDateString(lang, { month: 'short', day: 'numeric' });
    }

    // 跨年：月 日, 年（如 Jan 15, 2025）
    return target.toLocaleDateString(lang, { month: 'short', day: 'numeric', year: 'numeric' });
}
```

### 展示效果示例

| 时间差 | 英文 | 中文 |
|--------|------|------|
| < 1 分钟 | Just now | 刚刚 |
| 5 分钟 | 5m ago | 5 分钟前 |
| 2 小时 | 2h ago | 2 小时前 |
| 3 天 | 3d ago | 3 天前 |
| 同年内 | Jan 15 | 1月15日 |
| 跨年 | Jan 15, 2025 | 2025年1月15日 |

### 使用方式

```javascript
// 英文环境
formatDate('2026-04-18T10:30:00', 'en')  // "2h ago"
formatDate('2026-04-10T10:30:00', 'en')  // "Apr 10"
formatDate('2025-12-25T10:30:00', 'en')  // "Dec 25, 2025"

// 中文环境
formatDate('2026-04-18T10:30:00', 'zh')  // "2 小时前"
formatDate('2026-04-10T10:30:00', 'zh')  // "4月10日"
formatDate('2025-12-25T10:30:00', 'zh')  // "2025年12月25日"
```
