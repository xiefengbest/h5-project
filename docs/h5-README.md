# AnySelected H5 导读

本文件说明 **`pages-mobile/web-anyselected/index.html`** 资讯网站 Demo 的用途与配套文档。

| 场景 | 说明 |
|------|------|
| **资讯网站 Demo** | 列表 / 详情 / 说明页 / 多语言（单页多态） |
| **负一屏 v1** | `minus-one/news.html` 点击新闻 → 本文件 `?id=N` 详情 |
| **壁纸 App** | 详情页资讯入口 → 本文件 `?id=N` |

负一屏 v2 见 `pages-mobile/minus-one/discover.html`。

> 本项目为 **产品经理 / 设计用高保真原型**，本地打开 `pages-mobile/` 预览即可，不面向线上部署。

## 功能范围（Demo 已体现）

- **资讯列表**：分类筛选、条目列表（标题、来源、时间、封面等）
- **资讯详情**：正文（段落 + 配图）、相关推荐；非法 ID 的错误态
- **搜索**：按标题搜索，详情页可排除当前篇
- **多语言**：默认英文；切换后记忆偏好（URL 参数）
- **四则说明页**：无障碍、关于、隐私、服务条款
- **页脚**：版权回首页、四项说明入口

Demo 内数据为写死的示例；若嵌在 App 中，顶栏（返回、标题）可由原生壳提供，页面内可不重复实现。
